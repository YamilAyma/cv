# CV - Anthony Yamil Ayma Leon

[![Render CV](https://github.com/YamilAyma/cv/actions/workflows/render-cv.yml/badge.svg)](https://github.com/YamilAyma/cv/actions/workflows/render-cv.yml)

Este repositorio contiene mi CV gestionado como código utilizando [RenderCV](https://github.com/sinaatalay/rendercv).

## 🚀 Automatización

Este proyecto utiliza **GitHub Actions** para:
1. **Renderizar** automáticamente el CV en múltiples formatos (PDF, HTML, MD) cada vez que se actualiza el archivo YAML.
2. **Publicar** la versión web en [GitHub Pages](https://YamilAyma.github.io/cv/).
3. **Generar un Release** con el PDF actualizado.

## 📄 Formatos Disponibles

- **PDF**: [Descargar última versión](https://github.com/YamilAyma/cv/releases/latest)
- **Web**: [Versión online](https://YamilAyma.github.io/cv/)

## 🛠️ Tecnologías

- **Lenguaje**: YAML
- **Motor de Renderizado**: Typst vía RenderCV
- **Infraestructura**: GitHub Actions
- **Tipografía**: Charter
- **Tema**: sb2nov
